#FULL NAME
NovaScanner Professional Edition v2.0

##FULL DESCRIPTION
NovaScanner is a comprehensive security tool designed to intercept 
malicious scripts before they execute in your browser. It uses 
AI-driven heuristics to identify zero-day threats.

##PRE-REQUISITES
- Chrome Version 110 or higher
- 2GB Minimum RAM
- Active Internet Connection

##UTILITIES
- Real-time script blocking
- Automated vulnerability reporting
- One-click history cleaning
