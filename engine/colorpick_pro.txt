#FULL NAME
ColorPick Pro: Advanced Hex & RGB Eyedropper

##FULL DESCRIPTION
ColorPick Pro is a high-performance color eyedropper tool designed for 
web developers and digital artists. It allows for pixel-perfect color 
selection from any webpage with built-in history and palette generation.

##PRE-REQUISITES
- Chromium-based browser (Chrome, Edge, Brave)
- Hardware acceleration enabled for smooth zooming
- Screen recording permissions (for eyedropper functionality)

##UTILITIES
- Real-time color magnification
- Auto-copy to clipboard in HEX, RGB, or HSL
- Color palette history storage
- Contrast ratio checker for accessibility (WCAG)
